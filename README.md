## Technologies used:
#### Frontend:
-Html
#### FrameWork: 
- Asp.Net Core v2.1
#### IDE:
- Microsoft Visual Studio 2017 
#### Language:
- C#
#### S.O:

 -  Visual Studio 2017 or upper version
 -  .Net Core 2.1 or upper version.

1. Open Visual Studio 2017
2. Open the file "StarWars.sln"
3. Clean Solution
4. Build Solution
